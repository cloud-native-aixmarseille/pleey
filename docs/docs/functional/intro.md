# 🎮 Welcome to Pleey

Pleey is a community game platform, designed for real-time multiplayer experiences (quizzes and more).

## ✨ Features

- ✅ Multiple choice and true/false questions
- 👑 Complete admin interface
- 👥 User registration and authentication
- ⚡ Points based on accuracy AND response time
- 🏆 Live leaderboards with podium
- 🎨 Cyber Arcade design system (retro synthwave aesthetic)
- 🔄 Real-time gameplay with WebSockets
- 🌍 Multi-language support (English/French)

## 🎨 Design Philosophy

Pleey uses the **Cyber Arcade** design system, a retro-futuristic aesthetic that combines:

- 80s arcade gaming nostalgia
- Synthwave purple/pink/cyan color palette
- Terminal-style interfaces
- Pixel-perfect retro effects

See the [Design System](../technical/design-system) documentation for details.
