### User Story
As a player and admin, I want to see the results of a game in a visually appealing and shareable format, so that the experience is enjoyable and engaging.

### Acceptance Criteria
1. When the game is finished, display the results for the players.
2. Provide a shareable screen view of the results for the admin.
3. Include a fun, unique, and joyful cinematic/animation for displaying the results.

### Additional Notes
- Ensure the design aligns with the existing UI theme.
- The animation should be lightweight and performant.
- Consider accessibility to make the results view inclusive for all users.
